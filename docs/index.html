AngryBirds Framework

Un framework CSS personalizado inspirado en el universo de Angry Birds.
Construido con SASS, diseñado Mobile First.

🚀 Comenzar 📦 GitHub

🚀 Instalación

Elige la forma más conveniente para integrar AngryBirds Framework en tu proyecto.

⚡ Opción 1: Vía CDN

Agrega estos enlaces en el <head> de tu HTML:

<link rel="stylesheet" href="https://yani-m22.github.io/AngryBirds-FrameworkHDP/css/style.css"> <script src="https://yani-m22.github.io/AngryBirds-FrameworkHDP/js/navbar.js" defer></script>
📦 Opción 2: Descarga directa

Clona el repositorio y enlaza los archivos:

git clone https://github.com/Yani-M22/AngryBirds-FrameworkHDP.git

Luego enlaza en tu HTML:

<link rel="stylesheet" href="css/style.css"> <script src="js/navbar.js" defer></script>

📐 Grid System

Sistema de 12 columnas Mobile First. Usa .row y .col-* para estructurar tu layout.

Contenedores

.container
Ancho responsivo estándar, crece por breakpoints.
.container-fluid
Siempre 100% del ancho, sin límite.
.container-sm
Contenedor estrecho, ideal para formularios.
.container-lg
Contenedor amplio para layouts grandes.
<!-- Contenedor estándar responsivo --> <div class="container">...</div> <!-- Contenedor 100% del ancho --> <div class="container-fluid">...</div> <!-- Contenedor estrecho --> <div class="container-sm">...</div> <!-- Contenedor amplio --> <div class="container-lg">...</div>

Columnas

El sistema genera clases .col-1 a .col-12, .col-md-* y .col-lg-*.

🔴 Botones

Usa la clase base .btn combinada con una variante de estilo, un color y un tamaño.

Uso básico

Combina siempre 3 clases: .btn + variante + color + tamaño.

<button class="btn btn-solid btn-primary btn-md">Botón</button>

Variantes de estilo

Existen 4 variantes: .btn-solid, .btn-outline, .btn-ghost, .btn-pill.

<button class="btn btn-solid btn-primary btn-md">Solid</button> <button class="btn btn-outline btn-primary btn-md">Outline</button> <button class="btn btn-ghost btn-primary btn-md">Ghost</button> <button class="btn btn-pill btn-primary btn-md">Pill</button>

Variantes de color

Reemplaza btn-primary por cualquier color del framework.

<button class="btn btn-solid btn-primary btn-md">primary</button> <button class="btn btn-solid btn-danger btn-md">danger</button> <button class="btn btn-solid btn-warning btn-md">warning</button> <button class="btn btn-solid btn-success btn-md">success</button> <button class="btn btn-solid btn-dark btn-md">dark</button> <button class="btn btn-solid btn-light btn-md">light</button> <button class="btn btn-solid btn-accent btn-md">accent</button> <button class="btn btn-solid btn-pink btn-md">pink</button> <button class="btn btn-solid btn-gold btn-md">gold</button> <button class="btn btn-solid btn-pig btn-md">pig</button> <button class="btn btn-solid btn-silver btn-md">silver</button>

Tamaños

Usa .btn-sm, .btn-md o .btn-lg para controlar el tamaño.

<button class="btn btn-solid btn-primary btn-sm">Pequeño</button> <button class="btn btn-solid btn-primary btn-md">Mediano</button> <button class="btn btn-solid btn-primary btn-lg">Grande</button>

Estados

Los botones tienen estados hover, active y disabled integrados.

<!-- Hover y active son automáticos --> <button class="btn btn-solid btn-primary btn-md">Normal</button> <!-- Disabled --> <button class="btn btn-solid btn-primary btn-md disabled">Disabled</button>

⚠️ Alertas

Mensajes semánticos para comunicar éxito, peligro, advertencia o información. Combina .alert con .alert-{color}.

Estilos semánticos

Cada alerta tiene un título con .alert-title y un mensaje con .alert-message.

¡Éxito! La operación se completó correctamente.
¡Peligro! Ocurrió un error inesperado. Intenta de nuevo.
¡Advertencia! Esta acción no se puede deshacer.
¡Información! Tienes 3 mensajes nuevos en tu bandeja.
<!-- Éxito --> <div class="alert alert-success"> <strong class="alert-title">¡Éxito!</strong> <span class="alert-message">Operación completada.</span> </div> <!-- Peligro --> <div class="alert alert-danger"> <strong class="alert-title">¡Peligro!</strong> <span class="alert-message">Ocurrió un error.</span> </div> <!-- Advertencia --> <div class="alert alert-warning"> <strong class="alert-title">¡Advertencia!</strong> <span class="alert-message">Esta acción no se puede deshacer.</span> </div> <!-- Información --> <div class="alert alert-primary"> <strong class="alert-title">¡Información!</strong> <span class="alert-message">Tienes mensajes nuevos.</span> </div>

📝 Formularios

Estilización de inputs, selects y checkboxes con soporte para validación. Usa .form-group para agrupar cada campo con su label.

Estructura base

Siempre agrupa cada campo con .form-group, su .form-label y el input correspondiente.

<!-- Input --> <div class="form-group"> <label class="form-label">Nombre</label> <input type="text" class="form-input" placeholder="Tu nombre"> </div> <!-- Select --> <div class="form-group"> <label class="form-label">Opción</label> <select class="form-select"> <option>Opción 1</option> <option>Opción 2</option> </select> </div> <!-- Checkbox --> <div class="form-group"> <label class="form-label">Opciones</label> <label class="form-check"> <input type="checkbox"> Opción A </label> </div>

Tamaños

Usa .form-input-sm o .form-input-lg para variar el tamaño. Aplica igual para .form-select-sm y .form-select-lg.

<input type="text" class="form-input form-input-sm" placeholder="Pequeño"> <input type="text" class="form-input" placeholder="Estándar"> <input type="text" class="form-input form-input-lg" placeholder="Grande"> <select class="form-select form-select-sm">...</select> <select class="form-select form-select-lg">...</select>

Variantes de color

Agrega .form-input-{color} para cambiar el color del borde y acento en focus. Aplica igual para selects con .form-select-{color}.

<input type="text" class="form-input form-input-primary" placeholder="Primary"> <input type="text" class="form-input form-input-danger" placeholder="Danger"> <input type="text" class="form-input form-input-success" placeholder="Success"> <input type="text" class="form-input form-input-warning" placeholder="Warning"> <input type="text" class="form-input form-input-gold" placeholder="Gold"> <input type="text" class="form-input form-input-pink" placeholder="Pink"> <select class="form-select form-select-warning">...</select>

Estados de validación

Usa .is-valid o .is-invalid para indicar el estado del campo. Combínalos con .valid-feedback o .invalid-feedback para mostrar mensajes.

✅ ¡Correo válido!
❌ La contraseña es incorrecta.
✅ Selección válida.
<!-- Campo válido --> <div class="form-group"> <label class="form-label">Correo</label> <input type="email" class="form-input is-valid" placeholder="correo@ejemplo.com"> <span class="valid-feedback">¡Correo válido!</span> </div> <!-- Campo inválido --> <div class="form-group"> <label class="form-label">Contraseña</label> <input type="password" class="form-input is-invalid" placeholder="Tu contraseña"> <span class="invalid-feedback">La contraseña es incorrecta.</span> </div>

🛠️ Utilidades

Clases de un solo propósito para agilizar la maquetación sin escribir CSS extra.

Tipografía

Tamaños de fuente — usa .fs-1 a .fs-6:

fs-1 — 36px

fs-2 — 30px

fs-3 — 24px

fs-4 — 20px

fs-5 — 16px

fs-6 — 14px

<p class="fs-1">Texto grande</p> <p class="fs-2">Texto grande</p> <p class="fs-3">Texto mediano</p> <p class="fs-4">Texto mediano</p> <p class="fs-5">Texto normal</p> <p class="fs-6">Texto pequeño</p>

Grosor de fuente — usa .fw-light, .fw-regular, .fw-bold:

fw-light — Texto ligero

fw-regular — Texto normal

fw-bold — Texto negrita

<p class="fw-light">Texto ligero</p> <p class="fw-regular">Texto normal</p> <p class="fw-bold">Texto negrita</p>

Familias de fuente (Traidas desde Google Fonts) — usa .font-display, .font-accent, .font-body, .font-code:

font-display — Permanent Marker

font-accent — Jolly Lodger

font-body — Quicksand

font-code — Courier New

<p class="font-display">Permanent Marker</p> <p class="font-accent">Jolly Lodger</p> <p class="font-body">Quicksand</p> <p class="font-code">Courier New</p>

Alineación de texto — usa .text-left, .text-center, .text-right, .text-justify:

text-left — Alineado a la izquierda

text-center — Alineado al centro

text-right — Alineado a la derecha

text-justify — Texto justificado en ambos lados para que ocupe todo el ancho disponible del contenedor.

<p class="text-left">Izquierda</p> <p class="text-center">Centro</p> <p class="text-right">Derecha</p> <p class="text-justify">Justificado</p>
